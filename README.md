## **PySpark Weather Analysis**
### Самостоятельное задание №1 — Анализ погодных данных в PySpark

Этот проект посвящён анализу многолетних погодных наблюдений с разных метеостанций.
В рамках задания выполняются чтение данных, очистка, заполнение пропусков и аналитика в PySpark.

Данные включают следующие столбцы:

| Столбец        | Описание                                      |
|----------------|-----------------------------------------------|
| `station_id`   | ID метеостанции                               |
| `date`         | Дата наблюдения (формат YYYY-MM-DD)          |
| `temperature`  | Средняя температура в градусах Цельсия       |
| `precipitation`| Количество осадков в миллиметрах             |
| `wind_speed`   | Средняя скорость ветра в метрах в секунду    |
